KNN (K-Nearest Neighbors) is a **non-parametric, instance-based supervised learning algorithm**.  
It predicts the class of a new data point based on **similarity to existing labeled data**.

In simple terms:
> "A data point is classified by majority vote of its nearest neighbors."
